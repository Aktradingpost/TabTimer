# TabTimer - Smart Website Scheduler

Schedule websites to open automatically at specific times. Perfect for daily tasks, reminders, work workflows, and more!

## Features

### Free Features (Always Available)
- ✅ Schedule unlimited websites to open at specific times
- ✅ One-time and recurring schedules (daily, weekly, monthly, yearly)
- ✅ Extended repeat options (weekdays, biweekly, quarterly, etc.)
- ✅ Auto re-lock after opening (1-30 minutes)
- ✅ Search and filter schedules
- ✅ Keyboard shortcuts (Alt+L to schedule, Alt+O for management)
- ✅ Dark mode
- ✅ Export/Import backup
- ✅ Notifications before scheduled opens
- ✅ Auto-delete schedules by date (name with YYYY-MM-DD prefix)
- ✅ 2-minute grace period for missed schedules

## Installation

1. Download the extension ZIP file
2. Open Chrome and go to `chrome://extensions`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the extracted folder containing `manifest.json`

## Quick Start

1. **Schedule a page**: Press `Alt+L` on any webpage, or right-click and select "Schedule this page"
2. **Manage schedules**: Click the TabTimer icon → "Manage Schedules" or press `Alt+O`
3. **Try Premium**: Click "✨ Try Premium" button for a free 7-day trial

## Auto-Delete Feature

Name your schedule with a date prefix (YYYY-MM-DD) and it will automatically be deleted after that date:
- `2026-01-31 Project Deadline` → Deletes after Jan 31, 2026
- `2026-03-15 Monthly Report` → Deletes after Mar 15, 2026
